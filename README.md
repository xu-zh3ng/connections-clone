nyt style connections

add your own categories in the style:

{name: "your category name, color: "var(--difficulty)" (where difficulty is easy,medium,hard,extreme), words{"1,2,3,4"}} x4

and then add them to your own js file with an array named PUZZLE_ARCHIVE


default categories/words are from nyt june 12 2023
